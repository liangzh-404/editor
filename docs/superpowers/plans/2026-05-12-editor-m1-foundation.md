# Editor M1 Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first runnable macOS and iOS foundation for the local-first block editor: project scaffold, clean three-column/mobile shell, SQLite schema, repositories, and basic local page rendering.

**Architecture:** Use XcodeGen to create a small multi-target Xcode project with separate macOS and iOS app targets sharing Swift sources. SQLite is accessed through a focused database/migrator/repository layer. The app shows local repository data through a `WorkspaceViewModel` and does not depend on CloudKit in M1.

**Tech Stack:** Swift 6, SwiftUI, XCTest, SQLite3, XcodeGen-generated Xcode project, macOS 26.4 SDK, iOS 26.4 SDK.

---

## File Structure

- Create `project.yml`: XcodeGen spec for `EditorMac`, `EditorIOS`, and `EditorTests`.
- Create `Sources/EditorApp/EditorApp.swift`: shared SwiftUI app entrypoint for macOS and iOS targets.
- Create `Sources/EditorApp/AppEnvironment.swift`: app startup dependency wiring.
- Create `Sources/EditorCore/Models/EditorModels.swift`: workspace, page, block, and snapshot models.
- Create `Sources/EditorCore/Store/SQLiteDatabase.swift`: thin SQLite wrapper with explicit open/close/execute/query behavior.
- Create `Sources/EditorCore/Store/SchemaMigrator.swift`: schema version table and M1 tables.
- Create `Sources/EditorCore/Store/PageRepository.swift`: local bootstrap and page/block read APIs.
- Create `Sources/EditorCore/Features/Shell/WorkspaceViewModel.swift`: observable state for the shell.
- Create `Sources/EditorCore/Features/Shell/EditorShellView.swift`: responsive macOS three-column and iOS collapsed SwiftUI shell.
- Create `Sources/EditorCore/Observability/EditorLog.swift`: named OSLog categories for future instrumentation.
- Create `Tests/EditorTests/SchemaMigratorTests.swift`: migration tests.
- Create `Tests/EditorTests/PageRepositoryTests.swift`: repository bootstrap/read tests.
- Create `Tests/EditorTests/WorkspaceViewModelTests.swift`: shell view model tests.
- Create `Editor.xcodeproj`: generated by XcodeGen from `project.yml`.
